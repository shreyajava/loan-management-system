# Loan Management System

A full-stack Loan Management System built using Spring Boot, Spring Security, Thymeleaf, MySQL, Swagger, Docker, Git, and GitHub.

## Features

* Customer Management
* Loan Application Processing
* EMI Calculation
* Payment Tracking
* Dashboard Analytics
* Spring Security Authentication
* Swagger API Documentation
* Docker Containerization

## Tech Stack

### Backend

* Java 21
* Spring Boot
* Spring Security
* Spring Data JPA

### Frontend

* Thymeleaf
* HTML
* CSS

### Database

* MySQL

### Tools

* Swagger / OpenAPI
* Docker
* Git
* GitHub
* Maven

## Project Structure

* Customer Management Module
* Loan Management Module
* Payment Management Module
* Dashboard Module
* Security Configuration
* Exception Handling

## How to Run

1. Clone the repository.
2. Configure MySQL database credentials in `application.properties`.
3. Run the Spring Boot application.
4. Access the application at:

http://localhost:8085

## API Documentation

Swagger UI can be accessed after starting the application.
